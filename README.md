##  Overview

This project builds an **end-to-end sentiment analysis pipeline** to classify Amazon product reviews into **positive and negative sentiments** using NLP and machine learning.

> Converts raw customer feedback into **actionable business insights**.

---

##  Objectives

* Classify reviews into **positive / negative sentiment**
* Perform **EDA to uncover patterns**
* Compare ML vs Deep Learning models
* Deploy real-time prediction system

---

##  Workflow

```text
Raw Reviews → Preprocessing → EDA → Feature Engineering → Model → Deployment
```

---

##  Data Preprocessing

* NLTK-based cleaning & lemmatization
* Removed nulls, duplicates, noise
* Structured dataset for modeling 

---

##  Exploratory Data Analysis

* Sentiment distribution analysis
* Word frequency trends
* Word cloud visualization
* Relationship analysis (bivariate)

---

##  Model Development

| Model | Accuracy | ROC-AUC |
| ----- | -------- | ------- |
| SVM   | 91.90%   | 91%     |
| ANN   | 92.95%   | 90%     |

### Key Insight:

* ANN → better accuracy
* SVM → better efficiency
